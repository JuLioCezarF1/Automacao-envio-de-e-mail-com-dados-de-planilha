# Automacao-envio-de-e-mail-com-dados-de-planilha
Uma pequena automação onde todos os dados de uma planilha são tratados, e logo em seguida são transformados em corpo de um email e enviado.

A automação exige que a planilha que está anexada no repertório esteja na mesma pasta que o arquivo do código, assim a biblioteca pandas irá saber onde buscar os dados.
A automação foi pensada em uso exclusivo do Outlook versão clássica, que é o mais comum em empresas.
